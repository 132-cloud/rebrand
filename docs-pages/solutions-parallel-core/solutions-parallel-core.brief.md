# Creative Brief — Parallel Core

**Page:** /solutions/parallel-core
**In one line:** The Parallel Core page presents the "launch beside your core" path to market — five distinct use cases for institutions that need to grow without disrupting what they already run.

## What this page is for
Mid-funnel consideration for institutions that want to launch something new but can't touch their primary core. The page presents five parallel core use cases (vertical brands, new lines, M&A acceleration, private label, sandbox) as distinct section anchors. Primary action: book a demo.

## Who it's for
Primary: CEOs and Chief Strategy Officers at banks and credit unions who have a growth initiative — a new segment, an acquisition, a digital brand — stalled because it requires core involvement they can't afford. Secondary: CDOs and innovation leads exploring sandbox or test-and-learn models. They arrive knowing they have a growth idea; they don't know a parallel core is the mechanism.

## What we're communicating
- "Modernizing the core is one path. Launching beside it is the other." — this is the strategic frame for the whole page
- Five launch models on a parallel core: vertical brand, new line of business, M&A integration, private label, sandbox innovation
- Parallel deployment is independent — the primary core and its members are never touched
- Managed Services can run the operation so you don't have to staff it
- Three-card path callout bridges to the primary core page for institutions ready for a full replacement

## Tone & feel
Strategic and expansive, in the "Built for What's Next" brand chapter. This page is about possibility — five different growth paths unlocked by one architectural decision. The tone should feel like a strategic advisor pointing out options, not a features list. "One platform, two paths to market" is the governing frame.

## Must-haves / non-negotiables
- Five sequential split-panel feature blocks — each has a section ID for deep-linking from other pages (vertical-brands, new-lines, ma-acceleration, private-label, sandbox)
- Three-card path callout (New brands / Acquisitions / Innovation) with cross-link to core replacement
- Full FAQ (8 questions) verbatim — includes M&A and multi-brand scenarios
- Logo strip: "25 digital banks launched and cores deployed on Nymbus"
- Three testimonials: Sara Dolan (MSUFCU), Stephen Owen (First Entertainment CU/CineFi), Jill Castilla (Citizens Bank of Edmond)

## Interactive / visual elements
- Five split-panel sections: alternating text-left / text-right; all image slots currently placeholders — final designs should illustrate the distinct use case (e.g., brand identity for vertical brands, org chart / integration diagram for M&A)
- Section IDs (#vertical-brands, #new-lines, #ma-acceleration, #private-label, #sandbox) must be preserved for deep-linking from home page hero prompt routes and other pages

## What to avoid
- Don't collapse all five use cases into a generic "parallel core" benefits list — each use case has a distinct audience and should feel separately addressable
- Avoid implying parallel core is a temporary workaround before replacing the primary — it's a legitimate long-term strategy
- Em-dashes
